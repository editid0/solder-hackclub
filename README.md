# solder-hackclub

A simple letter r shaped PCB with some LEDs and a button that activates the LEDs, the variable resistor allows for dimming the LEDs if they are still too bright.

## Bill of materials:
- 1x Battery holder
- 1x Variable resistor
- 1x Button
- 2x LED
- 2x Resistor
